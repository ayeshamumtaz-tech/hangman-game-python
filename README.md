# 🎮 Hangman Game (Python)

A simple console-based **Hangman Game** developed in Python as a **Programming Fundamentals (PF) Lab Project**. The game allows players to guess randomly selected words from different difficulty levels while tracking their score and remaining attempts.

---

## 📌 Features

- 🎯 Three difficulty levels:
  - Easy
  - Medium
  - Hard
-  Random word selection
-  ASCII Hangman graphics
-  Input validation
-  Displays guessed letters
-  Score tracking
-  Play Again option
-  Game Over and  Winning messages

---

## 🛠️ Concepts Used

This project demonstrates the following Python concepts:

- Variables
- Lists
- Strings
- `while` loops
- `for` loops
- Conditional statements (`if`, `elif`, `else`)
- User input (`input()`)
- String methods:

  - `join()`

- Random module (`random.choice()`)

## ▶️ How to Run

1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:

```bash
python python_lab_project.py
```

---

## 🎮 How to Play

1. Select a difficulty level.
2. Guess one letter at a time.
3. Correct guesses reveal the letters in the hidden word.
4. Incorrect guesses reduce your remaining attempts.
5. Guess the complete word before all attempts are used.
